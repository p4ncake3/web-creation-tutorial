How to Make a Website with Github Pages and VS Code

Requirements

  • VS Code (Any Version)
  • A Github account
  • A computer/laptop (For VS Code)

Steps

  1. Open File Explorer (or any file browser) and create a new folder.
  2. Download and install VS Code then run it.
  3. Once inside VS Code, go to File > Open Folder and navigate to the folder you created in step 1. Press Select Folder.
  4. Next to the folder name (on the left side of VS Code) click the first icon with a partially folded paper and a + sign.
  5. Type index.html and press enter. (Optional: add style.css and script.js)
  6. Either ask ChatGPT to write your code OR handwrite it from scratch (or use a template).
  7. After writing your code, go to File > Save for each file.
  8. Go to Github and make sure you are logged in.
  9. Press the + button in the top right and select New Repository.
  10. Name your repository username.github.io (replace username with your Github username). This will be your user website.
  11. If making a project website, create a new repository with any name. The link will be https://username.github.io/example.
  12. Scroll down and tick Add README.
  13. Click Create Repository.
  14. In your repository, click Add Files > Upload Files.
  15. Click Choose Files, navigate to your folder, select all files, and click OK.
  16. Add a commit message like "Add website files" and press Commit Changes.
  17. Then go to the Settings tab and click Pages on the side. Then select None under Branches and change it to main.

Done! You can now visit your website at https://username.github.io or your project URL.
